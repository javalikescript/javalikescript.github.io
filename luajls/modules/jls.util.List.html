Module jls.util.List

Represents a list.

The functions could be used for Lua tables.

List.isName (value)
Returns true when the specified value is a Lua name. Names in Lua are any string of letters, digits, and underscores, not beginning with a digit and not being a reserved word.

Parameters:

  • value string The string value to check.

Returns:

    boolean true when the specified value is a Lua name.
List.isList (t[, withHoles[, acceptEmpty]])
Returns true when the specified table is a list. A list has continuous integer keys starting at 1. A list may have a field "n" giving the size of the list as an integer.

Parameters:

  • t table The table to check.
  • withHoles boolean true to indicate that the list may have holes. (optional)
  • acceptEmpty boolean true to indicate that the list could be empty. (optional)

Returns:

  1. boolean true when the specified table is a list.
  2. number the number of fields in the table or the number of elements in the list.
